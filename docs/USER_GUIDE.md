## 二、志愿匹配（核心页面）

左侧填信息，**任意修改都会即时刷新**右侧的冲/稳/保结果（约 0.3 秒）。所有内容保存在你本地浏览器。

### 各字段怎么填

| 字段 | 说明 |
| --- | --- |
| **省份** | 选你的高考省份。选省份后会**自动设置正确的选科模式与科类**（如山东自动变为「3+3 / 综合」）。带「暂无样本数据」的省份目前无数据。 |
| **城市** | 你所在城市，**选填**。**不影响冲/稳/保排名**，仅作为 AI 聊天的背景参考。 |
| **选科模式** | 新高考 3+3 / 新高考 3+1+2 / 传统文理。**一般随省份自动选好，不用手动改。** |
| **科类 / 首选** | 由选科模式决定：3+3→综合；3+1+2→物理或历史；传统→理科或文科。**山东应为「综合」。** |
| **选科组合** | 你选考的科目（多选）。用于专业选科匹配。 |
| **高考分数** | 必填。决定冲/稳/保的核心依据。 |
| **位次（可选）** | 若你知道自己的位次可直接填；留空则由分数自动换算（需该省一分一段表）。 |
| **兴趣方向** | **可多选**（点多个标签）；也可在下方输入框**自定义后回车**。 |
| **意向院校** | 逗号分隔，如「山东大学，中国海洋大学」。命中会在排序中加权（+20）。 |
| **意向专业** | 逗号分隔。**支持大类**（财会类→会计学/财务管理…）。**填了意向专业后，该校会按"你想读的那个专业自己的投档线"来判定冲/稳/保**，而不是按全校最低专业线——所以一所学校虽然最低线够得着，但若你想读的专业线更高、超出你的分数，它就不会被当作该专业的"保底"推荐（卡片会标 🎯 并显示专业线）。 |
| **偏好地区 / 城市层级** | 多选。命中的院校在排序中加权。 |
| **包含中外合作 / 高收费专业** | 复选框，**默认不勾选（排除）**。这类专业（中外合作办学/校企合作/较高收费）分数虚低但**学费昂贵（每年数万元）**，会把院校最低线拉低、制造"虚假保底"。默认排除；能负担学费可勾选。 |
| **其他个人想法** | 自由文本，发给 AI 作为参考（如「更看重就业 / 能接受调剂」）。 |

### 看懂结果卡片

结果分三档，每档**按"离你分数最近"排序**（最好的、最接近你分数的学校排在前面），**每档最多显示 40 所**，标题会显示"40 / 共 N"。想看某个具体学校（即使它在 40 名之外），用顶部**「院校查询」**直接搜。

- 🔴 **冲**：录取有挑战，建议服从调剂
- 🔵 **稳**：分数较匹配，志愿表的中坚
- 🟢 **保**：录取概率高，确保有学上

每张卡片含：

| 元素 | 含义 |
| --- | --- |
| 校名 + 城市 · 地区 | 院校与地理位置（部分院校暂无城市信息） |
| 标签 | 985 / 211 / 双一流（无标签显示为普通院校） |
| **预测线 vs 你的分** | 预测投档线与你的分数差（山东的预测线由真实「最低位次」经一分一段换算而来） |
| 位次 / 线 | 你的估算位次 与 该校最低位次 |
| **录取概率** | 基于分差（和位次）估算的概率，**仅为模型估计** |
| 转专业 | 转专业难度（容易/一般/较难/受限） |
| 专业 | 该校在你省份招生的专业列表（截断显示，如「等 41 个」） |
| **新鲜度徽章** | 数据较新 / 已 N 天未更新 / 往年数据 |
| 推荐理由 | 张雪峰式要点（985 平台、性价比、命中你的意向专业等） |

### 「数据提示」是什么意思

页面顶部若出现黄色提示，常见两种：

- **「缺少 XX·XX 当年一分一段表，位次为估算/缺失」**：该省份+科类没有一分一段表，位次只能估算或缺失。
- **「暂无 XX·XX 的院校投档线数据」**：该省份+科类**没有投档线数据**，所以出不了结果。

> 💡 **最常见原因：省份与科类不匹配。**
> 例如选了「山东 + 历史」——山东是 3+3，应为**综合**。把**选科模式改成「新高考 3+3」**（科类自动变「综合」）即可。

## 三、院校查询（按校名查分数）

想直接查某所学校的投档线（不依赖你的分数），用这个页面：

1. 顶部输入**院校名称**（支持模糊，如「山东大学」「海洋」「师范」）。
2. 选**考生省份 / 科类**（默认 山东 / 综合）。
3. 即时显示命中院校，每所含：
   - 标签（985/211/双一流）、城市、转专业难度
   - **历年投档线表**：年份 · 投档最低分 · 最低位次 · 新鲜度
   - 招生专业列表（截断显示）

> 💡 同一所学校可能出现多条（如「山东大学」「山东大学威海分校」「山东大学(中外合作)」）——这是**不同招生类型/校区**，分数线各不相同，属正常。

## 四、AI 聊天

- 顶部 **🐎 黑马预测** 横幅：展示"低分高校"机会，**带醒目免责声明：预测有风险,仅供参考,不可轻信**。
  （注：黑马识别需要多年投档数据，山东目前只有 2025 一年，暂不触发；可正常使用对话与方案功能。）
- **对话**：告诉 AI 你看重就业/城市/名气、风险偏好等，它会结合行业趋势给建议。
- **生成志愿方案**：一键产出 **稳妥型 / 均衡型 / 冲刺型** 三套完整方案（含理由与风险）。

> ⏳ **AI 较慢提示**：当前使用 DeepSeek 最强推理模型 **deepseek-reasoner (R1)**，会"深思考"，单次回复/方案可能需要 **20–60 秒**。想更快可改用 `deepseek-chat`（见下方「切换模型」）。未配置 API Key 时会回退到本地模拟回复。

## 七、常见问题 FAQ

**Q：山东填了没结果 / 提示缺数据？**
A：山东是 3+3，**科类必须是「综合」**。选省份「山东」后会自动设好；若手动改成物理/历史就会无数据。

**Q：CURRENT_YEAR 是 2025，现在不是 2026 吗？**
A：`CURRENT_YEAR` 指**数据所属招生年份**，不是日历年。我们的真实数据是 **2025 年**的，所以是 2025。改成 2026 会把 2025 数据误判为"往年数据"。等有 2026 数据时再改。

**Q：意向专业填大类（如财会类）能匹配吗？**
A：能。已支持大类→具体专业匹配（财会类→会计学/财务管理…）。

**Q：为什么每档都只有 40 所？为什么看不到某所学校（如齐鲁工业大学）？**
A：每档**只显示离你分数最近的前 40 所**（标题有"40 / 共 N"）。真实数据下一档可能有上千所，全列出来没法看。
某所学校若排在 40 名之外就不显示——这不代表它不在库里。**想查具体某校，用顶部「院校查询」直接搜**（齐鲁工业大学就在库中，2 个招生代号）。

**Q：中外合作专业分很低，会不会误导？**
A：会，所以**默认已排除**。中外合作办学/校企合作/较高收费专业分数虚低但学费昂贵（每年数万），
默认不计入院校最低线与意向专业匹配，避免"虚假保底"。能负担学费的可勾选「包含中外合作 / 高收费专业」。
（例：北京理工大学会计学(中外合作)628 分，但其平价专业最低约 661 分——默认按 661 评估。）

**Q：为什么填了财会类，某校（如宁夏大学）不出现 / 档位变高了？**
A：因为填了意向专业后，**按该校"你想读的那个专业自己的投档线"评估**，而不是全校最低线。
例如宁夏大学全校最低线是园艺（约 540），但它的会计学线更高——你 540 分能进宁夏大学（园艺），
但进不了它的会计学，所以它**不会**被当作"财会类"的保底。这是更诚实的结果（卡片有 🎯 标注与专业线）。

**Q：城市填了好像没用？**
A：城市**不参与冲/稳/保排名**，只作为 AI 聊天的背景。影响地理偏好的是「偏好地区 / 城市层级」。

**Q：AI 很慢 / 没反应？**
A：deepseek-reasoner 深推理较慢（20–60 秒），请耐心等待；想快用 deepseek-chat。若未配 Key 则为本地模拟回复。

**Q：录取概率/预测线准不准？**
A：预测线来自**真实官方数据**，但"概率/能否录取"是**模型估计**，今年存在波动，**仅供参考**。最终是否录取需以实际录取为准（见 `README.md` 的「数据校验」与 `docs/ROADMAP.md` 的「录取后追踪」）。

## 八、切换数据源 / AI 模型（进阶）

编辑项目根目录 `.env.local`：

```bash
# 数据源
DATA_SOURCE=real        # real=真实数据  mock=示例数据

# AI 模型
AI_PROVIDER=deepseek            # deepseek / anthropic / mock
DEEPSEEK_MODEL=deepseek-reasoner   # 最强(慢)；快用 deepseek-chat
# DEEPSEEK_API_KEY=...           # 你的 key（请勿提交到 git，已在 .gitignore）

# 当前招生年份（数据年份，不是日历年）
CURRENT_YEAR=2025
```

改完重启：`pnpm dev`。
